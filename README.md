# BullsCows (Быки-Коровы)
Приложение для игры в быки-коровы
[GitHub](https://github.com/pypi-zedzhen/BullsCows)

## Установка
> pip install BullsCows --index-url=https://pypi.zedzhen.ru/simple

## Запуск
> python -m BC

или
> BC

Далее будет использоваться 2ой вариант

## Запуск игры
> BC play
